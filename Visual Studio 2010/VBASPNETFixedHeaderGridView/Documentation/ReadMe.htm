VBASPNETFixedHeaderGridView

概述

在ASP.NET中GridView控件通常加载多行数据时包含垂直滚动条。当用户滚动使用垂直滚动条时, GridView的头将移动并消失。此示例说明了如何使用JQuery固定GridView头的位置。这里的演示可以在各种Web浏览器客户端上运行(IE, Firefox, Safari等)。

运行示例

请遵循下列演示步骤。

步骤1：打开VBASPNETFixedHeaderGridView.sln。

步骤2：展开VBASPNETFixedHeaderGridView web 应用程序按下  Ctrl + F5显示Default.aspx页面。

步骤3：你会看到页面上的GridView控件，请滚动垂直滚动条，GridView的标题行不会随滚动条移动。

步骤4：请使用其他浏览器浏览这个页面, 比如Chrome, FireFox, Safari, 等等。

步骤5：验证完成。

使用代码

步骤1：在Visual Studio 2010或Visual Web Developer 2010中创建一个VB.NET的 "ASP.NET 空白 Web 应用程序"。命名为"VBASPNETFixedHeaderGridView"。

步骤2：在应用程序的根目录下添加一个web页面, 命名为"Default.aspx"。

步骤3：创建一个文件夹命名为"JScript"，复制示例代码的JS文件到你自己的JS文件夹. 您还可以从这些网站下载:

        http://docs.jquery.com/Downloading_jQuery

        http://plugins.jquery.com/project/fixedtableheader

步骤4：在Default.aspx页面上添加一个GridView和添加一个新建数据表作为GridView的数据源。在这里我们给出Default.aspx.cs文件的代码。

       gvwList.DataSource = tab

步骤6：最后一步, 调用JQuery库编写JQuery函数固定GridView控件的头。

JQuery代码类似于这样：

- HTML code snippet -
 
<script src="JScript/jquery-1.4.4.min.js" type="text/javascript"></script><script src="JScript/ScrollableGridPlugin.js" type="text/javascript"></script>
        <script type = "text/javascript">
            $(document).ready(function () {
                // 调用 Scrollable 函数。
                $('#<%=gvwList.ClientID %>').Scrollable({
                ScrollHeight: 600,        
                });
            });
 
</script>
 
- end -
